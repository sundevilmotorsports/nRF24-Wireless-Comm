# Sender and Receiver code for the nRF24 wireless communication.

## Pinout for Arduino Leonardo
RF Receiver ----> Leonardo 

CE ----> 1 (TX1)

CS ----> 10

SCK ----> ICSP (SCK)

SDO ----> ICSP (MISO)

SDI ----> ICSP (MOSI)

GND

3.3V

## Pinout for Teensy

RF Transmitter ----> Teensy

CE ----> 2

CS ----> 10

SCK ----> 13

SDO ----> 12

SDI ----> 11

GND

3.3V
